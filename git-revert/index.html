Git 如何撤销分支的merge

首先回滚两种方式的区别

revert：是用一次新的commit回滚之前的commit,所以它的版本号不会变，这样就会导致在日后继续merge以前的老版本的时候，这些差异并不会显示，我们这次也有这个问题，master和develop对比差异的时候，并没有把回滚之后的差异显示出来。

reset：是直接删除指定的commit，提交及之前的commit都会被保留，但是此次之后的修改都会被退回到暂存区。
