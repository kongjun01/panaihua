setuptools д»Ӣз»Қ

setuptools жҳҜ distutils еўһејәзүҲпјҢдёҚеҢ…жӢ¬еңЁж ҮеҮҶеә“дёӯгҖӮе…¶жү©еұ•дәҶеҫҲеӨҡеҠҹиғҪпјҢиғҪеӨҹеё®еҠ©ејҖеҸ‘иҖ…жӣҙеҘҪзҡ„еҲӣе»әе’ҢеҲҶеҸ‘ Python еҢ…гҖӮеӨ§йғЁеҲҶ Python з”ЁжҲ·йғҪдјҡдҪҝз”Ёжӣҙе…Ҳиҝӣзҡ„ setuptools жЁЎеқ—гҖӮ

еҗ„дёӘеҸӮж•°зҡ„еҗ«д№ү

setup.py йңҖиҰҒзҡ„жЁЎеқ—

setup.py жү“еҢ…е‘Ҫд»Ө

жӯӨеӨ„еҲ—дёҫдёҖдәӣеёёз”Ёе‘Ҫд»Өпјҡ

  • build:

жһ„е»әе®үиЈ…ж—¶жүҖйңҖзҡ„жүҖжңүеҶ…е®№

  • build_ext:

жһ„е»әжү©еұ•пјҢеҰӮз”Ё C/C++, Cython зӯүзј–еҶҷзҡ„жү©еұ•пјҢеңЁи°ғиҜ•ж—¶йҖҡеёёеҠ  вҖ“inplace еҸӮж•°пјҢиЎЁзӨәеҺҹең°зј–иҜ‘пјҢеҚіз”ҹжҲҗзҡ„жү©еұ•дёҺжәҗж–Үд»¶еңЁеҗҢж ·зҡ„дҪҚзҪ®гҖӮ

  • sdist:

жһ„е»әжәҗз ҒеҲҶеҸ‘еҢ…пјҢеңЁ Windows дёӢдёә zip ж јејҸпјҢLinux дёӢдёә tag.gz ж јејҸ гҖӮжү§иЎҢ sdist е‘Ҫд»Өж—¶пјҢй»ҳи®Өдјҡиў«жү“еҢ…зҡ„ж–Үд»¶пјҡ

  • bdist:

жһ„е»әдёҖдёӘдәҢиҝӣеҲ¶зҡ„еҲҶеҸ‘еҢ…гҖӮ

  • bdist_egg:

жһ„е»әдёҖдёӘ egg еҲҶеҸ‘еҢ…пјҢз»Ҹеёёз”ЁжқҘжӣҝд»ЈеҹәдәҺ bdist з”ҹжҲҗзҡ„жЁЎејҸ

  • bdist_wheel:

жһ„е»әдёҖдёӘ wheel еҲҶеҸ‘еҢ…пјҢegg еҢ…жҳҜиҝҮж—¶зҡ„пјҢwhl еҢ…жҳҜж–°зҡ„ж ҮеҮҶ

  • install:

е®үиЈ…еҢ…еҲ°зі»з»ҹзҺҜеўғдёӯгҖӮ

  • develop:

д»ҘејҖеҸ‘ж–№ејҸе®үиЈ…еҢ…пјҢиҜҘе‘ҪеҗҚдёҚдјҡзңҹжӯЈзҡ„е®үиЈ…еҢ…пјҢиҖҢжҳҜеңЁзі»з»ҹзҺҜеўғдёӯеҲӣе»әдёҖдёӘиҪҜй“ҫжҺҘжҢҮеҗ‘еҢ…е®һйҷ…жүҖеңЁзӣ®еҪ•гҖӮиҝҷиҫ№еңЁдҝ®ж”№еҢ…д№ӢеҗҺдёҚз”ЁеҶҚе®үиЈ…е°ұиғҪз”ҹж•ҲпјҢдҫҝдәҺи°ғиҜ•гҖӮ
